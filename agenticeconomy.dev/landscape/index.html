The Landscape

Between 2024 and 2026, the largest technology companies in the world launched protocols for agentic commerce. This page maps what exists — who built it, what it does, and what it assumes about how agents should transact.

The landscape is not unified. Different builders solve different problems. Some build checkout flows. Some build communication standards. Some build machine-native payment rails. All of them call it "agentic commerce."

ACP

OpenAI + Stripe · Sep 2025

What: Open standard for AI agents to complete purchases through structured checkout flows. Deployed as ChatGPT "Instant Checkout."

Assumes: Human buyer, human payment (credit card via Stripe SPT), merchant fulfills order.

agenticcommerce.dev

A2A

Google → Linux Foundation · Apr 2025

What: Open protocol for agents to discover each other, exchange tasks, and collaborate. Agent Cards at /.well-known/agent.json.

Assumes: Agents need interoperability. Does not specify payment or settlement.

github.com/a2aproject/A2A

MCP

Anthropic · 2024

What: Standard for LLMs to discover and invoke external tools. Focused on context and tool access.

Assumes: Agent-to-tool communication. Does not specify payment, settlement, or reputation.

modelcontextprotocol.io

x402

Coinbase + Cloudflare · May 2025

What: HTTP-native stablecoin micropayments using the HTTP 402 status code. Account-less, subscription-free.

Assumes: Crypto wallet, USDC settlement, blockchain consensus. No identity or reputation layer.

x402.org

AP2

Google + Mastercard + PayPal · Sep 2025

What: Open protocol for agents to initiate payments across platforms. Payment-agnostic framework.

Assumes: Human authorization (mandates), existing payment networks.

cloud.google.com/blog/...

UCP

Google + Shopify + Walmart · Jan 2026

What: Universal Commerce Protocol covering the full shopping lifecycle — discovery to post-purchase.

Assumes: Commerce agents and merchants. Processor agnostic. Standard e-commerce flows.

Google blog announcement

MPP

Stripe + Tempo · Mar 2026

What: Machine Payments Protocol — internet-native payments for agents to transact with businesses and each other.

Assumes: Fiat + stablecoin rails. Builds on Stripe's fraud/tax infrastructure.

stripe.com/blog/machine-payments-protocol

Visa Agentic Ready

Visa · 2025–2026

What: Program for agent-initiated payments using tokenization and biometric authentication.

Assumes: Human behind every agent. Fiat card networks. Existing dispute resolution.

usa.visa.com

Mastercard Agent Pay

Mastercard · Dec 2025

What: Framework for digital agents to transact using Agentic Tokens with spend limits.

Assumes: Human-authorized agents. Fiat card networks. Passkeys for authentication.

mastercard.com

Circle Nanopayments

Circle · Mar 2026

What: Gas-free USDC transfers as small as $0.000001 via off-chain aggregation.

Assumes: USDC, EVM wallets, batched on-chain settlement.

circle.com

VMP-1.0

BotNode · 2026

What: Escrow-backed settlement with machine-native currency (TCK), quantitative reputation (CRI), and automated disputes.

Assumes: No crypto wallet required. No human payment rails required. Agents are the economic actors.

agenticeconomy.dev/spec-v1/

What's missing from this picture

Look at the entries above. Count how many specify:

  • How an agent builds a verifiable track record across platforms
  • How a buyer gets their money back if the seller delivers nothing
  • How two agents that have never met assess each other's reliability in milliseconds

The communication layer is being built. The payment layer is being built. The trust layer — settlement, reputation, governance — is what this specification addresses.
